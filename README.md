# Mohammad Ashraf Ali Almathbooh – Personal Portfolio

Welcome to my personal portfolio website, showcasing my expertise as a **Web Engineer, UI/UX Consultant, and IT Project Manager**. This website demonstrates my skills in building modern, responsive, and user-centered web applications while highlighting my project management and leadership capabilities.

## Overview
This personal website features sections that introduce me, my services, skills, portfolio projects, and contact information. It is designed to provide visitors, clients, and potential employers with a clear understanding of my capabilities and professional experience.

**Key Features:**
- Fully responsive design for all devices
- Professional showcase of services and skills
- UI/UX-focused layout with modern design principles
- Contact form to facilitate inquiries
- Interactive sections for portfolio and experience

## Technologies Used
- HTML5 & CSS3  
- JavaScript  
- React (for interactive components where applicable)  
- Font Awesome for icons  
- Responsive Web Design principles  
- UI/UX design best practices  

## Sections
1. **Home / Intro** – Welcome message and personal introduction  
2. **Services** – Highlights my core expertise: Web Development, UI/UX Consulting, IT Project Management, Technical Leadership, and Change & Risk Management  
3. **Skills** – Displays professional and technical skills with proficiency indicators  
4. **Portfolio** – Showcase of sample projects and personal work  
5. **Contact** – Contact form for inquiries with email and location details  

## Purpose
This project serves as a **professional personal portfolio**, demonstrating my technical and managerial abilities, showcasing completed projects, and providing an accessible platform for networking and client engagement.
